# 📊 Employee Analytics Dashboard (Excel Power Pivot)

## 🚀 Project Overview

This project presents a fully interactive **Employee Analytics Dashboard** built using Microsoft Excel. The dashboard transforms raw HR data into meaningful insights, enabling better understanding of workforce composition, salary trends, and employee experience.

The solution leverages **Power Pivot, DAX, PivotTables, PivotCharts, and Slicers** to create a dynamic and user-friendly reporting interface.

---

## 🎯 Key Objectives

- Analyse employee distribution across different dimensions  
- Understand salary patterns and trends  
- Evaluate employee satisfaction and work-life balance  
- Compare salary variations across experience levels and employment types  
- Build an interactive dashboard for real-time filtering and insights  

---

## 🛠️ Tools & Technologies

- Microsoft Excel  
- Power Pivot (Data Model)  
- DAX (Data Analysis Expressions)  
- PivotTables & PivotCharts  
- Slicers for interactivity  

---

## 📊 Dashboard Features

### 🔹 KPI Indicators
- Total Current Employees  
- Total Salary Paid  

### 🔹 Interactive Filters (Slicers)
- Gender  
- Department  
- Employment Type  

### 🔹 Visual Insights
- Employee distribution by gender  
- Education level breakdown (percentage-based pie chart)  
- Job satisfaction levels  
- Work-life balance ratings  
- Monthly salary trends (time-series analysis)  
- Salary comparison by years of service (Full-time vs Contractor)  
- Salary comparison by total working years and gender  

---

## 🧠 Key Skills Demonstrated

- Data modelling using Power Pivot  
- Writing DAX measures for business calculations  
- Data transformation and feature creation (Years of Service)  
- Building interactive dashboards in Excel  
- Data visualisation and storytelling

---

## 📈 Key Insights

- Salary trends show variation across months, indicating seasonal or organisational patterns  
- Full-time employees generally exhibit higher salary growth compared to contractors  
- Salary increases with experience, but trends vary across gender groups  
- Majority of employees fall within mid-to-high satisfaction and work-life balance categories  
- Education distribution highlights workforce qualification structure
